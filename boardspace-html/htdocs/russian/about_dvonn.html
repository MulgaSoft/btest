Dvonn

Dvonn is a stacking game where the number of stacks, and the mobility of the remaining stacks, is continously declining.  Like the other games in the Gipf series, it can be a real brain-burner - but it can also be played pretty casually.

Read the official rules on the project Gipf web site.

Robots:  Duimbot seems to be about right against beginners.   Smartbot and Bestbot were originally not much better, but their new, upgraded versions ought to be more challenging.
